How to use program

**To Compile:**

cd hw1_cli
ant dist

**To execute server:**

ant runComputer

*To execute client (in seperate terminal window:)**

ant -Darg0={ip_address_of_host_running_server} runMandelbrotClient
ant -Darg0={ip_address_of_host_running_server} runTSPClient

examples:

ant -Darg0=128.111.43.42 runMandelbrotClient (if runComputer is running on 128.111.43.42)
ant -Darg0=localhost runTSPClient (if runComputer is running on localhost)
