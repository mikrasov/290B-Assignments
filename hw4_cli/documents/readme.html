How to use program

**To Compile:**

cd hw2_cli
ant dist

**To Execute Space:**

ant -Darg0={numThreads} runSpace (use numTheads = 0 to disable space runnable tasks)

*To Execute Computer (in seperate terminal window:)**

If running space remotely:
ant -Darg0={remote_space_host_ip_address} -Darg1={prefetchBufferSize} -Darg2={numThreads} -Darg3={enableCaching} runComputer (where enableCaching is either true or false)

**To Execute Client (in seperate terminal window:):**

ant -Darg0={spaceDomain} -Darg1={fibItteration} runFibClient

ant -Darg0={spaceDomain} runTspClient
