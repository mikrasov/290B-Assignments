How to use program

**To Compile:**

cd hw2_cli
ant dist

**To Execute Space:**

ant -Darg0={numLocalThreads} runSpace

*To Execute Computer (in seperate terminal window:)**

ant -Darg0={remote_space_host_ip_address} -Darg1={desiredPrefetchBufferSize} -Darg2={desiredNumThreads} runComputer

**To Execute Client (in seperate terminal window:):**

ant -Darg0={remote_space_host_ip_address} -Darg1={12 or 16 cities} -Darg2={true or false branchAndBound} runTspClient
