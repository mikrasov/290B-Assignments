How to use program

**To Compile:**

cd hw3_cli
ant dist

**To Execute Space:**

ant runSpace

*To Execute Computer (in seperate terminal window):**

If running space locally but in separate jvm:
ant -Darg0=localhost -Darg1={computer_id} runComputer

If running space remotely:
ant -Darg0={remote_space_host_ip_address} -Darg1={computer_id} runComputer

**To Execute Client (in seperate terminal window. Note: -Darg2 is only used for the Fib Client):**

If running all three components on the same jvm:
ant -Darg0=jvm -Darg1={num_computers} -Darg2=16 runFibClient (or runTspClient)

If running space locally but in separate jvm:
ant -Darg0=localhost -Darg1=-1 -Darg2=16 runFibClient (or runTspClient)

If running space remotely:
ant -Darg0={remote_space_host_ip_address} -Darg1=-1 -Darg2=16 runFibClient (or runTspClient)
