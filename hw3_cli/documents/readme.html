How to use program

**To Compile:**

cd hw3_cli
ant dist

**To Execute Space:**

ant runSpace

*To Execute Computer (in seperate terminal window:)**

If running space locally but in separate jvm:
ant -Darg0=localhost -Darg1={computer_id} runComputer

If running space remotely:
ant -Darg0={remote_space_host_ip_address} -Darg1={computer_id} runComputer

**To Execute Fibonacci Client (in seperate terminal window:):**

If running space locally but in separate jvm:
ant -Darg0=localhost -Darg1={fibonacci_number_to_get} runFibClient

If running space remotely:
ant -Darg0={remote_space_host_ip_address} -Darg1={fibonacci_number_to_get} runFibClient

**To Execute Traveling Salesman Client (in seperate terminal window:):**

If running space locally but in separate jvm:
ant -Darg0=localhost runTspClient

If running space remotely:
ant -Darg0={remote_space_host_ip_address} runTspClient
